## Git Components

### Git Repository

- It contains files, history, and configuration managed by Git.
- It is simply your project folder where Git tracks all changes.

👉 When you run: `git init` - git it turns your normal folder into a repository.

---

### .git Folder

Inside every Git repository, there is a hidden folder called `.git`.

👉 This folder:

- Stores all your project history  
- Keeps track of commits, branches, and configurations  
- To prevent accidental deletion
- If you delete this folder, your entire Git history will be lost.
---

### Remote Repository

- It is an online storage for your code.
- Examples: GitHub, GitLab  

👉 Purpose

- Backup your code  
- Share with team members  
- Collaborate from anywhere  
---

## Summary

- **Repository** = your project folder  
- **.git folder** = hidden folder
- **Remote repository** = online storage for your code
