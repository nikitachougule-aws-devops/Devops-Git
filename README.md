# Git Basics

A beginner-friendly Git learning repository that covers Git fundamentals, branching, merging, GitHub workflows, troubleshooting, interview preparation, and AI-assisted Git practices.

## 📚 Course Structure

```text
git-basics/
│
├── 01. git introduction.md
├── 02. git components.md
├── 03. git stages.md
├── 04. git workflow.md
├── 05. git commands.md
├── 06. AI with Git.md
├── 07. git branching and merging.md
├── 08. pull request (PR).md
├── 09. git rebase, restore, reset, revert.md
├── 10. git troubleshooting.md
├── 11. git project.md
├── 12. git interview questions.md
├── 13. git interview troubleshooting.md
```

## 🎯 What You'll Learn

* Git fundamentals
* Git architecture and components
* Git workflow
* Git commands
* Branching and merging
* Pull Requests (PR)
* Rebase, Restore, Reset, and Revert
* Git troubleshooting
* AI-assisted Git workflow
* Real-world Git project
* Git interview questions
* Git interview troubleshooting

## 📖 Learning Path

1. Learn Git basics.
2. Practice common Git commands.
3. Understand branching and merging.
4. Learn collaboration using Pull Requests.
5. Practice troubleshooting common Git issues.
6. Complete the Git project.
7. Revise interview questions.
8. Use AI to improve your Git workflow.

## 🎯 Learning Outcome

After completing this repository, you will be able to:

* Use Git confidently for version control.
* Collaborate with teams using GitHub.
* Resolve merge conflicts and common Git issues.
* Manage branches and Pull Requests.
* Recover from common Git mistakes.
* Use AI effectively to improve your Git workflow.
* Answer Git interview questions with confidence.
